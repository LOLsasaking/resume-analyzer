# Resume Analyzer

A privacy-first resume analyzer that scores structure, contact details, sections, action verbs, measurable outcomes, weak phrases, and writing style directly in the browser.

## Live Demo

https://resume-analyzer-rouge-omega.vercel.app

## Features

- Instant score with ten focused resume checks
- Strong-action-verb and measurable-impact analysis
- Contact, portfolio, experience, education, and skills detection
- Weak phrase and first-person writing checks
- Built-in realistic example
- No uploads, account, backend, API key, or data collection

## Local Use

Open `index.html` directly or serve the directory:

```bash
python3 -m http.server 8000
```

## Technology

HTML, CSS, and JavaScript with a deterministic analysis engine.
